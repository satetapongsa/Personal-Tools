# 🚀 PT Dash - Personal Tools & Dashboard

**PT Dash** (Personal Tools Dashboard) คือศูนย์รวมเครื่องมือและข้อมูลส่วนตัวที่ออกแบบมาเพื่อเพิ่มประสิทธิภาพในการทำงาน (Productivity) และการติดตามผลลัพธ์ในด้านต่างๆ ไม่ว่าจะเป็นการลงทุน การเรียนรู้ด้าน Cybersecurity หรือเครื่องมือทาง Network

## ✨ Features (ความสามารถหลัก)

### 📊 Assets & Crypto Tracker
- ติดตามราคาเหรียญ Crypto และกองทุนรวม (Mutual Funds) แบบ Real-time
- แสดงเปอร์เซ็นต์การเปลี่ยนแปลงและกราฟสรุปเบื้องต้น

### 🛡️ Cybersecurity Study Tracker
- ระบบติดตามความคืบหน้าการเรียนรู้ (Learning Roadmap)
- บันทึกคะแนนและสถานะการเตรียมสอบ (เช่น CCNA, Security+, PenTest+)

### 🌐 Networking Tools
- **Subnet Calculator**: เครื่องมือคำนวณ IP Subnetting ที่แม่นยำและรวดเร็ว
- สนับสนุนการคำนวณ CIDR, Wildcard Mask และ Host Range

### 📅 Personal Schedule
- จัดการตารางเรียน ตารางสอบ และนัดหมายสำคัญ
- ระบบแจ้งเตือนกิจกรรมที่กำลังจะมาถึง

## 🛠️ Tech Stack (เทคโนโลยีที่ใช้)

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Outfit (Google Fonts)

---

## 🚀 Getting Started (เริ่มต้นใช้งาน)

1. **Clone the repository**
   ```bash
   git clone https://github.com/satetapongsa/Personal-Tools.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

## 📝 Commit Standard

โปรเจกต์นี้ใช้ระบบ **Conventional Commits** เพื่อความเป็นมืออาชีพ:
- `feat:` สำหรับฟีเจอร์ใหม่
- `fix:` สำหรับการแก้ไข Bug
- `docs:` สำหรับการแก้ไขเอกสาร
- `style:` สำหรับการปรับปรุงดีไซน์ (CSS)
